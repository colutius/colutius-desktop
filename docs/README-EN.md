## Build manually
- Clone the repository
```bash
git clone --recursive https://github.com/colutius/colutius-desktop.git
cd colutius-desktop
```
- Start build
```bash
mkdir build
cd build
cmake ..
make
```
- Run the program
```bash
./colutius-desktop
```
