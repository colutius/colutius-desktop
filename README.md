## 手动构建
- 克隆仓库
```bash
git clone --recursive https://github.com/colutius/colutius-desktop.git
cd colutius-desktop
```
- 开始构建
```bash
mkdir build
cd build
cmake ..
make
```
- 运行软件
```bash
./colutius-desktop
```
